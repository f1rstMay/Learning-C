一、算术操作符

+-*/%

1.+ - * / 操作符可以作用于整数和浮点数；

2.%操作符只能用于两个整数之间，返回的是整除之后的余数；

3./ 操作符则是根据两操作数类型进行除法，如两个操作数是整型，执行整数除法，如果两数为浮

点型，则执行浮点数除法。

二、移位操作符

<< 左移操作符>> 右移操作符

1.左移操作符

移位规则： 左边抛弃、右边补0

2.右移操作符

右移运算分两种：

（1）逻辑移位 ：左边用0填充，右边丢弃

（2）算术移位 ：左边用原该值的符号位填充，右边丢弃

注意！注意！⚠     不能移动负数位！

三、位操作符

&|^

&：按位与（有0为0）

| ：按位或（有1为1）

^：按位异或（相同为0，不同为1）

（位运算是通过补码进行计算的）

  1. #include <stdio.h>
  2. int main()
  3. {
  4.  int num1 = 1;
  5.  int num2 = 2;
  6.  int a,b,c;
  7.  a = num1 & num2;
  8.  b = num1 | num2;
  9.  c = num1 ^ num2;
  10.  return 0;
  11. }

  1. int x = 0;

复合赋值符 

+=*=%=<<=|=
-=/=>>=&=^=
  1. int x = 10;
  2. x = x+10;
  3. x += 10;

!逻辑反操作

 ！（逻辑反操作）：

  1. int a = 0;
  2. while(!a)
  3. {
  4. //语句
  5. }
  6. //!a为真，while进入死循环

 &（取地址）：

  1. int p=1;
  2. int *a=NULL;
  3. a=&p;

 sizeof （操作数的类型长度（以字节为单位））：

  1. int a = 10;
  2. printf("%d\n", sizeof(a));//长度为4字节
  3. printf("%d\n", sizeof(int));//长度为4字节
  4. printf("%d\n", sizeof a);//长度为4字节
  5. printf("%d\n", sizeof int);//写法错误，求类型长度不能将括号去掉

~（对一个数的二进制按位取反）：

 前置++和 --

  1. #include <stdio.h>
  2. int main()
  3. {
  4.     int a = 10;
  5.     int x = ++a;
  6.     //先对a进行自增，然后对使用a，也就是表达式的值是a自增之后的值。x为11。
  7.     int y = --a;
  8.     //先对a进行自减，然后对使用a，也就是表达式的值是a自减之后的值。y为10;
  9.     return 0;
  10. }

后置++和 --

  1. #include <stdio.h>
  2. int main()
  3. {
  4.     int a = 10;
  5.     int x = a++;
  6.     //先对a先使用，再增加，这样x的值是10；之后a变成11；
  7.     int y = a--;
  8.     //先对a先使用，再自减，这样y的值是11；之后a变成10；
  9.     return 0;
  10. }

八、条件操作符

exp1 ? exp2 : exp3

exp1是判断条件，为真执行exp2，为假执行exp3

九、逗号表达式

exp1, exp2, exp3, …expN

逗号表达式，就是用逗号隔开的多个表达式。

逗号表达式，从左向右依次执行。整个表达式的结果是最后一个表达式的结果。

  1. int a = 1;
  2. int b = 2;
  3. int c = (a>b, a=b+10, a, b=a+1);
  4.              //a=12      b=12+1=13

十、下标引用、函数调用和结构成员

1.[ ] 下标引用操作符

操作数：一个数组名 + 一个索引值

  1. int arr[10];//创建数组
  2.  arr[6] = 1;//实用下标引用操作符。
  3.  //[ ]的两个操作数是arr和6。

2.( ) 函数调用操作符

接受一个或者多个操作数：第一个操作数是函数名，剩余的操作数就是传递给函数的参数。

  1. void test()
  2. {
  3.     printf("Lumos\n");
  4. }
  5.  
  6. int main()
  7.  {
  8.  test();  //实用（）作为函数调用操作符。
  9.  return 0;
  10.  }

3.访问一个结构的成员

. 结构体.成员名

-> 结构体指针->成员名

  1. #include <stdio.h>
  2. struct Stu
  3. {
  4.  char name[10];
  5.  int age;
  6.  char sex[5];
  7. };
  8.  
  9. int main()
  10. {
  11.     struct Stu s = {"zhansan", 20, '男'};
  12.     struct Stu* p = &s;
  13.  
  14.     printf("%s %d %c\n",s.name,s.age,s.sex);
  15.     printf("%s %d %c\n",p->name,p->age,p->sex);
  16. //以上两句所表达的含义均是结构成员访问
  17.     return 0;
  18. }
